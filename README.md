# 🚀 Prepify – A Smart Way to Learn

Prepify is a React-based interview preparation platform designed to help students prepare for coding interviews, aptitude tests, and technical interviews through a clean and interactive user interface.

## ✨ Features

### 🏠 Home Page

* Modern and responsive landing page
* Hero section
* Features section
* Student feedback section
* Professional footer

### 💻 DSA Practice

* Search questions
* Filter by topic
* Filter by difficulty
* Curated DSA questions
* Direct links to solve questions

### 🧠 Aptitude Preparation

* Quantitative Aptitude
* Logical Reasoning
* Verbal Ability
* Data Interpretation
* Multiple-choice questions
* Automatic score calculation

### 🎤 Mock Interview

* Frontend Developer Interview Questions
* Backend Developer Interview Questions
* Full Stack Developer Interview Questions
* Next/Previous Question Navigation
* Interview Practice Environment
